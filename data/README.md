# Data

## Overview
This folder contains sample datasets used for development, testing, and demonstration of the data pipeline.

---

## Contents

- **abs_regional_population_*.json**  
  Regional population data from the Spatial Urban Data Observatory (SUDO)

- **pedestrian-counting-system-sensor-locations.json**  
  Locations of pedestrian sensors across Melbourne

- **demo.csv**  
  Example dataset for testing and validation

---

## Notes

- These datasets are **sample or partial extracts** of larger real-world data sources  
- Full datasets are accessed dynamically via APIs or external platforms  
- Data is used for **testing pipeline functionality and visualisations**
